# Actividad: Configuración de interfaz WAN con DHCP

## Objetivo

Configurar la interfaz `ether1` de un router para que obtenga automáticamente una dirección IP mediante un servidor DHCP y comprobar que el router tiene acceso a Internet.

## Enunciado

Configura el router siguiendo uno de los tutoriales proporcionados en clase para que la interfaz `ether1` funcione como cliente DHCP.

Una vez realizada la configuración, comprueba que el router obtiene correctamente todos los parámetros necesarios para acceder a Internet.

## Tareas a realizar

1. Configurar la interfaz `ether1` como cliente DHCP.
2. Verificar que el router recibe una dirección IP válida.
3. Comprobar que existe una ruta por defecto.
4. Verificar los servidores DNS configurados.
5. Realizar una prueba de conectividad a Internet.

## Entrega

Entregar un documento breve o capturas de pantalla donde se muestre:

- La opción elegida (tutorial seguido)
- La configuración de red aplicada al router
- Las rutas configuradas
- Los servidores DNS configurados
- Una prueba de conectividad realizando un ping a google.es

## Requisitos

- La configuración debe ser funcional.
- Las evidencias deben ser claras y legibles.
