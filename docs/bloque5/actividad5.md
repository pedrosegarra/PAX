# En esta actividad deberás entregar las evidencias que demuestren el correcto funcionamiento de las configuraciones realizadas esta semana:

- Segmentación de red mediante VLAN.
- Routing inter-VLAN.
- Configuración del servicio DNS en el router.
- Forzado del uso del servicio DNS mediante redirección.

La finalidad es verificar que el entorno configurado funciona de forma coherente e integrada. Deberás entregar un único documento o evidencias gráficas que muestren:

- La segmentación VLAN operativa.
- La comunicación entre VLAN a través del router.
- La asignación correcta de direccionamiento en cada VLAN.
- El funcionamiento del servicio DNS configurado en el router.
- El correcto funcionamiento del mecanismo de forzado del servicio DNS, incluso cuando un cliente intenta utilizar un servidor externo.
