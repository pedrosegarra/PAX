# Introducción a MikroTik y RouterOS

---

## ¿Qué es MikroTik?

### Filosofía

MikroTik propone una forma de entender las redes desde dentro.

- Permite observar el funcionamiento
- Permite intervenir en la red
- Evita automatismos opacos

El objetivo no es solo que funcione, sino entender por qué funciona.

---

### Ecosistema

MikroTik dispone de distintos tipos de dispositivos:

- Routers pequeños para aprendizaje
- Equipos profesionales
- Dispositivos inalámbricos
- Switches

Todos comparten:

- Mismo sistema (RouterOS)
- Misma lógica de funcionamiento

## MikroTik frente a otros fabricantes

### Cisco

- Orientado a grandes empresas
- Alto coste
- Certificaciones estructuradas

### MikroTik

- Más accesible
- Más flexible
- Ideal para aprendizaje

---

## RouterOS: el corazón del sistema

RouterOS es el sistema operativo de MikroTik.

Define:

- Comportamiento del router
- Servicios disponibles
- Función dentro de la red

Un mismo dispositivo puede ser:

- Router
- Firewall
- Punto de acceso
- Switch lógico

---

## ¿Qué permite hacer RouterOS?

### Interconectar redes

- Enrutamiento
- Acceso a Internet

### Controlar tráfico

- QoS
- Limitación de ancho de banda

### Proteger la red

- Firewall
- Filtrado

### Servicios

- DHCP
- DNS

### Redes inalámbricas

- WiFi
- Control de clientes

### Monitorización

- Uso de recursos
- Análisis de tráfico

---

## Formas de interactuar con RouterOS

RouterOS puede administrarse mediante:

### Interfaces gráficas

- WinBox
- WebFig

### Línea de comandos

- SSH
